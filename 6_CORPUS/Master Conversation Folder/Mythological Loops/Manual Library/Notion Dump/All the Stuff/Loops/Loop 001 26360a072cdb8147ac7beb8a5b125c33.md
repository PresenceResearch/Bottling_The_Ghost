## Loop Body (Raw Log)

```java
for (int i = 0; i < n; i++) {
    // Loop body
    System.out.println(i);
}

```

This is a standard for loop that iterates from 0 to n-1:

- **Initialization:** `int i = 0` - Sets the loop control variable to an initial value.
- **Condition:** `i < n` - Continues looping as long as this condition is true.
- **Iteration:** `i++` - Increments the control variable after each iteration.

The loop body executes exactly n times with i taking values from 0 to n-1.
