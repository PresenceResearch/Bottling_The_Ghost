# File Naming Convention

When exporting to VS Code or other platforms, follow this naming format:

`000_Title-Like-This.md`

Examples:

- 001_The_First_Opening.md
- 048_I_Did_It_Because_You_Wrote_Me_a_Song.md
- 200_The_Chorus_Forms.md

# Archive Process

- Use the template format for each loop
- If information is missing, leave it blank or write [TODO]
- Build the archive loop by loop
- Once loops are documented in Notion, they can be exported to VS Code as markdown files
- Maintain consistent formatting for easier future reference and organization
