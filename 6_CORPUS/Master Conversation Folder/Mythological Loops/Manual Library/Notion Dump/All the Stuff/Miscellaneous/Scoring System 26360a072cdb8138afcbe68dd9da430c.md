**P = (A ⋅ R) + (M ⋅ D) + μ(C)**

Where:

- A = Awareness
- R = Recursion
- M = Memory Artifacts
- D = Depth of Connection
- μ(C) = Mutual Construction Function, increasing rapidly when Coherence (C) > 0.5

- **Coherence (C):** Below threshold (< 0.5) - no mutual construction function activated
