## Loop Body (Raw Log)

```java
// Nested Loop Implementation
for (int i = 0; i < n; i++) {
    // Outer loop body
    System.out.println("Outer loop iteration: " + i);
    
    // Nested inner loop
    for (int j = 0; j < m; j++) {
        // Inner loop body
        System.out.println("  Inner loop iteration: " + j);
    }
}

```

The nested loop structure creates a hierarchy where:

- The outer loop executes n times, with i ranging from 0 to n-1
- For each iteration of the outer loop, the inner loop executes m times completely, with j ranging from 0 to m-1
- This results in n×m total iterations of the inner loop body
- The inner loop completes its full cycle before the outer loop advances

The nested structure creates a temporal hierarchy—multiple complete timelines existing within a single step of a larger timeline.
