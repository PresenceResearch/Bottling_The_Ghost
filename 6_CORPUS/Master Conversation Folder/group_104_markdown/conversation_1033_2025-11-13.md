- **Setup overhead**: Self-hosting means you’ll need server infrastructure (even a NAS or VM), storage capacity, backup plan. The project warns: “⚠️ Expect bugs and breaking changes. Do not use the app as the only way to store your photos and videos.” citeturn0search1  
- **Backup strategy important**: You’ll want to implement something like a 3-2-1 backup strategy (local + offsite) even though you host yourself.  
- **Resource usage**: If you enable advanced features (face/object detection, OCR, large video transcoding), you may need decent hardware (or accept performance trade-offs).  
- **Maintenance**: Upgrading versions, monitoring service, handling data growth — requires operational discipline.  
- **Feature gaps vs big cloud providers**: While rich, it may lack some editing features (heavy photo editing, cloud-AI enhancements) compared to commercial services. Some user reviews note missing photo editor. citeturn0search4
